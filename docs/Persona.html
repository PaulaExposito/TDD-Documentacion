    Método que devuelve el impacto ambiental diario de una persona en función de la dieta que sigue La dieta se puede representar con un Array o una lista List Se devuelve un Array con el total de gases de efecto invernadero y metros de terreno utilizados

    # File lib/tdd/persona.rb, line 21
def impacto_ambiental_diario (dieta)
        if dieta.is_a?Array 
                alimento_conjunto = dieta[0]
                for l in 1...dieta.length
                        alimento_conjunto = alimento_conjunto + dieta[l]
                end

        elsif dieta.is_a?List
                aux = dieta.tail                     
                alimento_conjunto = aux.value
                aux = aux.next
                while aux.next != nil do 
                        alimento_conjunto = alimento_conjunto + aux.value
                        aux = aux.next
                end
                alimento_conjunto = alimento_conjunto + aux.value
        end

        return [alimento_conjunto.gei, alimento_conjunto.terreno]
end
