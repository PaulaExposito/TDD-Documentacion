    module TDD

    Módulo principal del proyecto

    version.rb¶ ↑

    El módulo TDD contiene las constantes de alimentos que se utilizan en las pruebas. Contiene el número de proteínas, carbohidratos, lípidos, gases de efecto invernadero y terrero

    También contiene la versión del programa
